# HZNU-3D Dataset



## 📚 Dataset Introduction

HZNU-3D is a large-scale, high-density campus Airborne Laser Scanning (ALS) point cloud dataset designed for remote sensing point cloud semantic segmentation research.

### ✨ Dataset Features

| Attribute | Value |
|-----------|-------|
| **Total Points** | Approximately 660 million points |
| **Data Format** | Standard LAS format |
| **Point Dimensions** | 6 dimensions (x, y, z, R, G, B) |
| **Annotation Method** | Fine-grained per-point annotation |
| **Annotation Quality** | Multiple rounds of manual calibration |
| **Scene Type** | Complex university campus scenes |

**Detailed Description**:

- **Scene Coverage**: Includes typical ground object categories such as buildings, vegetation, trees, playgrounds, and ground surfaces
- **Data Characteristics**: Integrates spatial coordinates and color information to comprehensively characterize geometric structures and semantic information in complex campus scenes
- **Application Scenarios**: Addresses the challenges of overlap and occlusion in campus scenarios, providing high-quality data support for few-shot semantic segmentation research
- **Annotation Accuracy**: Achieves fine-grained per-point annotation through multiple rounds of manual calibration, ensuring precise semantic boundaries for all categories
